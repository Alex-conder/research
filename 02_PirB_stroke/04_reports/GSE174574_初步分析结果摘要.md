# GSE174574 初步分析结果摘要

## 分析流程
1. 读取 6 个 10x 样本（3 Sham + 3 MCAO，24 h），共 **58,528 个细胞**。
2. 质控（n_genes ≥ 500, total_counts ≥ 1000, pct_mt < 20%）后保留 **58,287 个细胞**。
3. normalize_total + log1p 标准化。
4. 高变基因 → PCA → Neighbors → UMAP → Leiden 聚类（21 个簇）。
5. 基于已知 marker 进行细胞类型注释：
   - Astrocyte（Aqp4, Gja1）
   - Microglia（P2ry12, Tmem119）
   - Immune（Ptprc, Lyz2, Cd68）
   - Endothelial（Cldn5, Ly6c1）
   - Pericyte（Pdgfrb）
   - Oligodendrocyte（Mog, Mbp）
   - Ependymal（Foxj1）

## 主要结果

### 1. Pirb 在 MCAO 后显著上调
| 条件 | Pirb 阳性率 | 平均表达（log1p norm） |
|------|------------|----------------------|
| Sham | 2.7–2.8%   | 0.040–0.043          |
| MCAO | 8.2–9.8%   | 0.119–0.137          |

MCAO 后 Pirb 阳性率约为 Sham 的 **3 倍**，平均表达约为 Sham 的 **3 倍**。

### 2. Pirb 表达的细胞类型分布（横向扩展）
| 细胞类型 | MCAO Pirb+ % | Sham Pirb+ % | 结论 |
|----------|-------------|-------------|------|
| Immune | **31.37%** | 24.24% | **Pirb  highest**，以外周浸润免疫细胞为主 |
| Microglia | **6.16%** | 4.59% | 小胶质细胞 Pirb 表达显著，MCAO 后升高 |
| Ependymal | **4.41%** | 0.88% | 室管膜细胞在 MCAO 后 Pirb 明显上调 |
| Astrocyte | **3.13%** | 0.66% | 星形胶质细胞 Pirb+ 比例升高约 5 倍 |
| Oligodendrocyte | 2.52% | 0.92% | 少突胶质细胞少量表达 |
| Pericyte | 1.83% | 0.61% | 周细胞少量表达 |
| Endothelial | 1.77% | 0.58% | 内皮细胞少量表达 |

### 3. 关键发现
- **Pirb 并非仅在星形胶质细胞中表达**；在本数据集中，**免疫细胞（Ptprc+）和小胶质细胞的 Pirb 阳性率最高**。
- 星形胶质细胞中 Pirb+ 细胞比例从 Sham 的 0.66% 升至 MCAO 的 3.13%，升高约 **4.7 倍**，支持原报告关于 Pirb+ 星形胶质细胞的结论。
- 该结果提示：缺血后 Pirb 的上调可能具有更广泛的细胞类型参与，尤其在**免疫/炎症细胞**中。
