### 1.1 GSE174574（原报告主数据集）
| 项目 | 内容 |
|------|------|
| GEO 登录号 | GSE174574 |
| 标题 | Single-cell RNA-seq reveals the transcriptional landscape in ischemic stroke |
| 状态 / 公开日期 | Public on Sep 24, 2021 |
| 物种 | *Mus musculus*（C57BL/6J） |
| 模型 | 大脑中动脉闭塞（MCAO） |
| 分组 | 3 例 MCAO vs 3 例 Sham |
| 采样时间 | MCAO / Sham 后 24 h |
| 测序平台 | 10x Genomics Chromium；Illumina；GPL21103 |
| 原始数据 | `GSE174574_RAW.tar`（约 263 MB） |
| 本地路径 | `D:\Pirb_stroke_project\01_raw_data\GSE174574\` |
| 样本数 | 6 个样本，10x 标准 3 文件格式（barcodes.tsv.gz / features.tsv.gz / matrix.mtx.gz） |
| 相关链接 | BioProject: PRJNA730565；SRA: SRP320164 |
| 发表文献 | Zheng K, Lim L, Jiang W, et al. J Cereb Blood Flow Metab. 2022;42(1):56-73. PMID: 34496660 |

**审查结论**：
- 该数据集为 10x Genomics 3' 单细胞转录组，质控后原报告保留 57,769 个细胞、27,998 个基因；其中星形胶质细胞 5,813 个。
- 数据集仅含 24 h 单一时间点，无空间坐标，无法回答 Pirb+ 细胞出现的时间动态和空间定位。
- 该数据集的注释显示除星形胶质细胞外，还包含内皮细胞、单核/巨噬细胞、室管膜细胞、小胶质细胞、少突胶质细胞、周细胞/血管周围细胞、淋巴细胞、粒细胞、神经前体细胞等，具备横向扩展至其他细胞类型的基础。

## 五、数据库质量与劣势审查

| 审查维度 | 结论 | 对策 |
|----------|------|------|
| 时间维度 | GSE174574 仅 24 h 单时间点 | 引入 GSE227651（1/3/7 d）和 GSE225948（2/14 d） |
| 空间维度 | 无空间信息 | 引入 GSE233815 空间转录组；补充免疫荧光/RNAscope |
| 细胞覆盖 | 原报告仅分析星形胶质细胞 | 横向扩展至小胶质细胞、内皮细胞、少突胶质细胞、免疫细胞、神经元 |
| 样本量 | 3 vs 3，统计效力有限 | 多数据集整合（Harmony/Scanorama）扩大样本量 |
| Pirb 表达丰度低 | 阳性细胞比例低，差异分析受限 | 采用更宽松的表达阈值、伪 bulk 整合、独立队列验证 |
| 细胞身份存疑 | Pirb+ 星形胶质细胞高表达免疫/髓系基因 | 严格质控：排除 doublet、环境 RNA、免疫细胞污染 |
| 机制深度 | 仅描述相关差异基因 | 纵向深挖配体-受体、信号通路、转录因子调控网络 |

## 六、建议的数据整合与分析路线

1. **数据整合**：使用 scanpy / Seurat + Harmony 整合 GSE174574、GSE227651、GSE225948 的免疫细胞/内皮细胞子集。
2. **细胞注释**：使用 SingleR + 已知 marker（Aqp4/Gja1 星形胶质细胞；P2ry12/Tmem119 小胶质细胞；Cldn5/Ly6c1 内皮细胞；Mog/Mbp 少突胶质细胞；Rtn1/Sox4 神经元等）。
3. **Pirb 表达分析**：按细胞类型计算 Pirb 阳性率、平均表达；比较 MCAO vs Sham、不同时间点、不同空间区域。
4. **横向细胞扩展**：分别提取小胶质细胞、内皮细胞、少突胶质细胞、神经元、免疫细胞亚群，进行 Pirb+ vs Pirb- 差异分析。
5. **纵向机制深挖**：
   - 配体-受体分析：Nogo-A/MAG/OMgp → Pirb；MHC I → Pirb；IL-1β/TNF/C3 → 诱导 A1 星形胶质细胞。
   - 信号通路：RhoA-ROCK、SHP-1/2、NF-κB、JAK-STAT、NLRP3。
   - 细胞通讯：CellChat / CellPhoneDB 分析 Microglia/Immune/Neuron → Astrocyte 的通讯。
6. **功能验证假设**：提出 Pirb+ A1-like 星形胶质细胞可能由小胶质细胞来源的 IL-1α/TNF/C1q 诱导，并通过 Pirb 信号进一步放大炎症反应。
